Brackets & performers

Add one category per performance group (weight class, gender division, age range). Add performers (name and team) under each category. Performance order is the order categories are listed: the top category is 1st in the queue, then 2nd, 3rd, and so on. Use the ↑ / ↓ arrow buttons to change that order.

Team order (queue)

List teams in the order they should appear in the performer queue (1st team, then 2nd, then 3rd, …). Within each performance category, order follows this rotation. Names must match the Team field on performers (spelling).

Performance & judging

Each judge enters a score from 1 up to this maximum (default 10). Use any cap your event needs (e.g. 10, 20, 100)—up to 1000 here. Averages and finals use these values after deductions.
